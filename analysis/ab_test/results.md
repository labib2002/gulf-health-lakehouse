# A/B Test Results — notification template A vs B

> Generated by `analysis/ab_test/ab_test.py`. **Synthetic data.**

## Setup

- **Unit:** user. **Variant:** A = control template, B = new template.
- **Outcome:** `engaged_next_day` (binary: opened/acted the next day).
- **Primary test:** two-proportion z-test (binary outcome). Welch t-test reported
  as a cross-check.
- **alpha = 0.05** (two-sided), **target power = 0.8**,
  **designed MDE = 3%** absolute lift.

## Hypotheses

- **H0:** engagement_rate(B) = engagement_rate(A)  (the new template has no effect)
- **H1:** engagement_rate(B) ≠ engagement_rate(A)

## Observed

| Variant | Users (n) | Engaged | Rate |
|---|---:|---:|---:|
| A (control)   | 249 | 46 | 18.47% |
| B (treatment) | 251 | 50 | 19.92% |
| **Difference (B − A)** | | | **+1.45%** |

## Results

- **two-proportion z-test: effect=+0.0145  stat=0.411  p=0.6814  95% CI=(-0.0546, +0.0835)**
- **welch t-test: effect=+0.0145  stat=0.410  p=0.6821  95% CI=(-0.0549, +0.0838)**
- Decision at alpha=0.05: **FAIL TO REJECT H0 — not significant**.

The z-test and Welch t-test agree (expected: with a 0/1 outcome and these sample
sizes the two are asymptotically equivalent).

## Sample size & power (this is the important part)

To detect the designed **3%** absolute lift at alpha=0.05,
power=0.8, the normal-approximation formula requires
**≈ 2,787 users per group** (5,574 total).

This experiment actually ran with **249 per group**, giving an achieved
power of only **≈ 13%** for a 3% effect. So a non-significant
result here is **inconclusive, not evidence of no effect** — the study is
underpowered by design (the synthetic universe has 500 users). With the required
~2,787/group, the same true effect would be detectable.

See `docs/img/ab_power_curve.png`.

## Threats to validity (what would have invalidated this)

- **Underpowered:** the dominant issue here — too few users to detect a small
  lift, so absence of significance ≠ absence of effect.
- **Peeking / optional stopping:** running the test repeatedly and stopping when
  p<0.05 inflates the false-positive rate. The analysis is run once, post-hoc.
- **Assignment imbalance / SRM:** if the A/B split deviated from 50/50 beyond
  chance (sample-ratio mismatch), randomization is suspect. Here n_A≈n_B.
- **Outcome leakage / contamination:** if a user could see both templates, or the
  "next-day engagement" window overlapped the assignment, the effect would be
  biased. The synthetic design assigns one variant per user.
- **Novelty effect:** a new template can spike engagement transiently; a longer
  horizon than "next day" would test durability.
